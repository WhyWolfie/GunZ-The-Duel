# GunZ .mrs Tools

Tools for unpacking / packing `.mrs` files in GunZ: The Duel.

## Standard (normal) .mrs
- **MRSDecompiler Ultimate**  
  https://github.com/WhyWolfie/GunZ-The-Duel/tree/master/tutorial/mrs/MRSDecompiler%20Ultimate  
  → Best for beginners, works with original GunZ files + basic custom encryption

## Encrypted (protected) .mrs
Two main options:

1. **Pro MRS GUI** (recommended)  
   https://github.com/WhyWolfie/GunZ-The-Duel/tree/master/tutorial/mrs/pro_mrs_gui  
   → Easy GUI, create encrypted files, make your own encryptor exe

2. **GunnyCrypt** (older method)  
   https://github.com/WhyWolfie/GunZ-The-Duel/tree/master/tutorial/mrs/gunnycrypt  
   → Classic way with tutorial (also has video guide)

Backup your files first!
