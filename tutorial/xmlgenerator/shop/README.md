# GunZ: The Duel – Shop.xml Generator Tool

Quick & simple tool to convert your entire **zitem.xml** into a functional **Shop.xml** (adds basically all items to the in-game shop automatically).

**Original tool author:** Steven Corndog / StevenUGG (early contributor to EuroGunZ → Universe Gamers GunZ era)

## Download

📦 **[Shop Maker.zip](https://github.com/WhyWolfie/GunZ-The-Duel/blob/master/Tools/xmlgenerator/Shop%20Maker.zip)**

## How to Use

1. **Extract and launch**  
   Unzip the file → double-click **Shop Maker.exe**  
   ![Main window](https://i.imgur.com/Sh2rUVj.png)

2. **Open your zitem.xml**  
   Menu: **File → Open**  
   ![File → Open dialog](https://i.imgur.com/bqeXxRB.png)

3. **Select zitem.xml**  
   Browse to your GunZ files (client or server side), choose **zitem.xml** and open it.  
   ![Selecting zitem.xml](https://i.imgur.com/Sz64k8w.png)

4. **Wait for processing**  
   The tool reads and loads every item — can take a few seconds to minutes depending on file size.  
   Be patient until it finishes! 🙃  
   ![Loading items](https://i.imgur.com/IfEfjWc.png)

5. **Save the result**  
   Menu: **File → Save**  
   It will generate **Shop.xml** (usually in the same folder or prompts for location).  
   ![Saving Shop.xml](https://i.imgur.com/0uCWQju.png)

6. **Place the file correctly**  
   Copy the new **Shop.xml** into your **MatchServer** folder  
   → **directly** in the MatchServer directory (NOT inside system.mrs).  
   ![Correct placement](https://i.imgur.com/JsjeBuZ.png)
