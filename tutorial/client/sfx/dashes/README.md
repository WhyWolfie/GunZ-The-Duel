# 🎨 Custom Dash Colours – GunZ: The Duel

Change your dash trail colour in a few steps.

## How to do it

1. Go to → [`Colour Dash SFX`](https://github.com/Durial06/GunZ-The-Duel-Items/tree/main/dash)  
   Pick the colour you want

2. Decompile `SFX.mrs`  
   (with any MRS unpacker)

3. Replace files  
   Copy everything from the chosen dash folder  
   → overwrite in the extracted SFX folder

4. Repack `SFX.mrs`

5. Place it in your GunZ folder

Done! 💨

**Tip:** Always backup your original `SFX.mrs` first.

Enjoy your new dash colour!
