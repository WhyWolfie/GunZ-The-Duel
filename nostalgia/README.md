# GunZ: The Duel Archive

Classic third-person shooter (2003–) — wall-running, sword dashes, K-style combos.  
Preserving old official pages, interviews, launchers, and early private servers via Wayback Machine snapshots.

> **Note**: All links are archived captures. Many original sites are long gone.  
> No active downloads or servers hosted here — purely historical preservation.

## Highlights

- **Developer Interview**  
  [mrkim07 Interview (4gamer, 2004)](https://www.4gamer.net/specials/2004summer/mrkim/mrkim07.html)
